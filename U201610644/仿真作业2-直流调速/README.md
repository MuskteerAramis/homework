### 二. 单闭环调速系统算法仿真
##### 1. 确定增益值

采用纯比例P控制，改变比例增益，使得速度阶跃响应超调量为5%和20%，并记录对应比例增益值Kp1和Kp2。

以下为计算超调量matlab函数脚本：

```
%计算超调
y = temp_motn.values*9.55;
x = temp_mott.time;

ymax = max(y);
l = length(x);
ysta = mean(y(500:1000));

over = ymax/ysta-1;
over

```
得到相应的比例增益： 
kp1 = 0.0003 超调量为 9.44%   
kp2 = 0.0086 超调量为20.01%
